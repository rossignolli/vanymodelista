O esquema de modelagem de hoje é o traçado de uma manga, que você pode usar para vários tipos de mangas.

Existem diversas formas de traçar uma manga e essa é uma das que mais gosto. Se adequando a qualquer tipo de manga em tecido plano.

Para esse tipo de manga, recomenda-se que ela tenha uma pequena folga para fazer o embebido. Cerca de 2 a 3cm costuma funcionar para a maioria dos tecidos, mas essa folga pode chegar a até 4cm dependendo do tecido que será utilizado.

Para iniciar, o molde da roupa para a qual a manga será traçada precisa estar pronto, com todas as alterações que envolvem a cava, como alargamentos de ombro e folgas de vestibilidade.
